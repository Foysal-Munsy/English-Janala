Frequently Asked Questions

The difference between var, let, and const
***var can be updated and redeclared
***let can be updated and cannot be redeclared
***const cannot be updated and redeclared
The difference between map(), forEach(), and filter()
***map() return new array with modified value
***forEach() doesn't return anything
***filter() returns a new array with elements that pass a condition
Explain arrow functions and how they are different from regular functions
Arrow functions provide a shorter syntax and inherit this from the surrounding scope, while regular functions have their own this. Additionally, arrow functions can't be used as constructors and don't have their own arguments object.
How JavaScript Promises work?
JavaScript Promises represent the eventual completion (or failure) of an asynchronous operation and allow chaining of .then() for success or .catch() for error handling.
How closures work in JavaScript write it in one line?
Closures in JavaScript allow a function to retain access to variables from its outer lexical scope, even after the outer function has executed.
